Options

/help
Alias: /?

Displays this help message. Use in conjunction with /verbose for more detailed information.

If this switch is used with any switches other than /verbose then those switches will be ignored, the help message will be printed, and 0 will be returned as the process exit code.

/html

Causes help to be output as HTML.

Must be used with the /help switch.

/quiet
Alias: /q

Quiet mode - no output.

/verbose
Alias: /v

Verbose mode.

/database:<database>
Alias: /db

The name of the database to create a snapshot of.

/username:<username>
Alias: /u

The username for the database.

If no username is specified then integrated security is used.

/password:<password>
Alias: /p

The password for the database.

You must also provide a username. If you do not specify a username and password combination then integrated security is used.

/server:<server>
Alias: /s
Default: (local)

The name of the SQL server.

/makesnapshot:<makesnapshot>
Alias: /mksnap

Creates a snapshot of the database with the path specified.
